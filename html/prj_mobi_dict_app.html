Mobile Dictionary App

a project at PINGCOM, From 2012

About

A Chinese dictionary mobile application for Nokia feature phones.

Roles

  • J2ME developer
  • Dictionary data structure & algorithm designer

Leveraged knowledge in

  • Utilizing J2ME to build a Chinese dictionary app that support multi Chinese input methods
  • Getting familiar with Chinese bitmap fonts
  • Using StarDict (an open dictionary format) to develope Chinese data sources for the application
  • Developing an algorithm to reduce index list of the dictionary to fit on memory of low-end phones
  • Extending the knowledge for developing a Korean dictionary mobile application (a Samsung VietNam exclusive application on Bada phones)
