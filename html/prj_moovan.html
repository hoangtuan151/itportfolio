Moovan

a startup project at PINGCOM, Jan 2016 - Mar 2016

About

An Uber-Like platform that connects trucks and parcel's owners.

Roles

  • Design database for some sub-systems (Trucks Finding, Routes Tracking);
  • Code the above sub-systems;
  • Deploy the system.

Leveraged knowledge in

  • Getting farmiliar with MySQL free-schema design technique (like Uber's Mezzanine and Pinterest) ;
  • Utilizing Google Cloud Compute Engine for hosting backend system;
  • Leveraging Google's S2 Geometry for finding nearby trucks (see why);
  • Using GMap API for Price Calculation Module.
